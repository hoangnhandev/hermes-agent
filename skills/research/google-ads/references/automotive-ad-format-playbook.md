**Last Updated:** 2026-06-30

## Responsive Search Ads (RSA)

### RSA Structure Requirements

**Minimum Requirements:**
- **Headlines:** Minimum 3, maximum 15 (30 characters each)
- **Descriptions:** Minimum 2, maximum 4 (90 characters each)
- **Paths:** Optional, 0-2 (15 characters each)

**Best Practice:**
- **Headlines:** Use all 10-15 slots (more combinations = better optimization)
- **Descriptions:** Use all 4 slots (A/B testing at scale)
- **Pinning:** Pin brand name to position 1, CTA to position 2

### Pinning Strategy

**Pin Brand Name (Position 1):**
- Forces brand/model to always appear first
- Increases brand recognition
- Example: `[VF3]: Xe Điện Giá Rẻ Nhất` (pinned position 1)

**Pin CTA (Position 2):**
- Forces call-to-action to always appear
- Increases click-through rate
- Example: `Đặt Lái Thử Miễn Phí Hôm Nay` (pinned position 2)

**Don't Over-Pin:**
- Pin only 2-3 headlines maximum
- Leave others unpinned for Google optimization
- Test different pinning combinations

### Path Strategy (0-2 Paths)

**VF3 Campaign Example:**

**Path 1:**
- `/xe-dien` (category)

**Path 2:**
- `/vf3-gia-re` (model + benefit)

**Display URL:** `vinfast.vn/xe-dien/vf3-gia-re`

**Path Best Practices:**
- Use descriptive, keyword-rich paths
- Keep paths under 15 characters each
- Include model names + benefits

## Ad Extensions (Critical for CTR Boost)

### 1. Sitelink Extensions (Mandatory: 4 Links)

**Sitelink Requirements:**
- Minimum 2, maximum 6 sitelinks
- Link text: Maximum 25 characters
- Description lines: Maximum 35 characters each (2 lines)

**VF3 Campaign Sitelinks:**

**Sitelink 1: Book Test Drive (Primary CTA)**
- **Link Text:** `Đặt Lái Thử`
- **Description Line 1:** `Lái thử miễn phí VF3`
- **Description Line 2:** `Không cam kết mua xe`
- **Destination URL:** `/vf3/test-drive-booking`

**Sitelink 2: Price List (Informational)**
- **Link Text:** `Bảng Giá`
- **Description Line 1:** `VF3 từ 240 triệu VNĐ`
- **Description Line 2:** `So sánh các mẫu xe`

### 4. Location Extension (Critical for Dealers)

**Location Extension Requirements:**
- Google My Business account linked
- Dealer addresses verified
- Business categories: "Car Dealer", "Electric Vehicle Charging Station"

**Setup:**
1. Link Google My Business account to Google Ads
2. Verify dealer locations (showroom, service center)
3. Set location extensions to show nearest dealer

**Benefits:**
- Shows dealer address on ads
- "Visit" button for directions
- "Call" button for phone calls
- Distance to nearest dealer displayed

**Expected CTR Lift:** +10-15% with location extensions

### 5. Call Extension (For Phone Support)

**Call Extension Requirements:**
- Google forwarding number (tracks calls)
- Business hours (show only during open hours)
- Call conversion tracking (optional but recommended)

**Setup:**
1. Enable call extensions in campaign settings
2. Use Google forwarding number (tracks calls)
3. Set business hours (9AM-6PM Vietnam time)
4. Enable call reporting (duration, conversion)

**Expected CTR Lift:** +5-8% with call extensions

**Conversion Tracking:**
- Count calls >60 seconds as conversions
- Assign conversion value ($50-75 per qualified call)
- Feed into smart bidding (tCPA)

### 6. Image Extension (Visual Appeal)

**Image Extension Requirements:**
- High-quality vehicle images (white background preferred)
- Image dimensions: 600x314 or 1200x628
- File size: Max 5MB
- Format: JPG, PNG, non-animated GIF

**VF3 Image Strategy:**
1. **Exterior shot:** VF3 front 3/4 view (white background)
2. **Interior shot:** Dashboard/seating (clean, bright)
3. **Lifestyle shot:** VF3 in urban setting (Vietnam context)
4. **Charging shot:** VF3 at charging station

**Expected CTR Lift:** +10-15% with image extensions

## Performance Max Vehicle Ads (2025+)

### Performance Max Requirements

**Setup Requirements:**
1. **Google Merchant Center Account**
   - Create account at merchants.google.com
   - Verify website domain
   - Link to Google Ads account

2. **Vehicle Feed**
   - Required fields: VIN, make, model, year, price, condition, image, mileage
   - Optional fields: color, transmission, fuel type, vehicle options
   - Feed format: Google Sheets, XML, or API upload
   - Update frequency: Daily (price changes, inventory updates)

3. **Landing Pages**
   - Vehicle detail pages (one per model)
   - Mobile-optimized design
   - Fast loading speed (<3 seconds)
   - Clear CTAs (Book Test Drive)

### Vehicle Feed Structure

**Required Fields (VF3 Example):**

| Field | Value | Example |
|-------|-------|---------|
| **ID** | Unique identifier | `VF3-2024-BLUE` |
| **Title** | Vehicle title | `VinFast VF3 Mini EV` |
| **Make** | Manufacturer | `VinFast` |
| **Model** | Model name | `VF3` |
| **Year** | Model year | `2024` |
| **Price** | Selling price | `240000000` (VND) |
| **Condition** | Vehicle condition | `new` |
| **Image Link** | Main image URL | `https://vinfast.vn/images/vf3-front.jpg` |
| **Mileage** | Odometer reading | `0` (new vehicle) |

**Optional Fields:**

| Field | Value | Example |
|-------|-------|---------|
| **Color** | Exterior color | `Xanh Dương` |
| **Transmission** | Transmission type | `Tự động` |
| **Fuel Type** | Fuel/Power type | `Điện` |
| **Vehicle Options** | Features list | `10 năm bảo hành, thuê pin, sạc nhanh` |
| **Vehicle ID** | VIN (if available) | `VF3-VN-123456` |

**Campaign Structure:**

**Campaign Name:** `PMax-Vehicle-Ads-VF3-2024`

**Budget Allocation:**
- Start: 20-30% of total monthly budget

**Geographic Targeting:**
- Match dealer network coverage

**Logos (1):**

**Headlines (5-10):**
- `VinFast VF3 - Mini EV`

- `280km cho mỗi lần sạc`
- `10 năm bảo hành chính hãng`

**Descriptions (4-5):**

### Performance Max Optimization

**Learning Period (Weeks 1-4):**
- Do not make significant changes
- Let Google optimize creative combinations
- Monitor performance metrics
- Build conversion data (aim for 30+ conversions)

**Scaling Period (Weeks 5-8):**
- Add new creatives based on learnings
- Adjust geographic targeting
- Scale budget by 20-30% if ROAS positive
- Exclude low-performing demographics

**Mature Phase (Weeks 9+):**
- Test new audiences (similar audiences)
- Expand to new geographies
- Increase budget to 30-40% of total
- Feed conversion data back into optimization

**Performance Max Metrics:**

| Metric | Target | Action |
|--------|--------|--------|
| **Conversion Rate** | >8% | Scale winners, pause losers |
| **CPC** | <$2.50 | Optimize feed quality |
| **ROAS** | >3:1 | Increase budget |
| **Impression Share** | >60% | Increase bids/budget |

## Ad Copy Best Practices

### Character Limits

**Responsive Search Ads:**
- **Headlines:** Maximum 30 characters each
- **Descriptions:** Maximum 90 characters each
- **Paths:** Maximum 15 characters each
- **Display URL:** Auto-generated from final URL + paths

**Performance Max:**
- **Headlines:** Maximum 30 characters each
- **Descriptions:** Maximum 90 characters each
- **Long headlines:** Maximum 90 characters (optional)

### Do's and Don'ts

**DO:**
- ✅ Use all available headline/description slots
- ✅ Include model name in headlines
- ✅ Include numbers (280km, 10 năm, 240 triệu)
- ✅ Include CTAs (Đặt lái thử, Báo giá, So sánh)
- ✅ Test different message angles (range, price, warranty)

**DON'T:**
- ❌ Use all caps (VIETNAM instead of Vietnam)
- ❌ Use excessive punctuation (!!!, ???)
- ❌ Make absolute claims (#1, best ever, cheapest)
- ❌ Use competitor brand names (unless conquest campaign)
- ❌ Repeat same message across all headlines

## Ad Testing Strategy

### A/B Testing Framework

**Test 1: Message Angle (4 Weeks)**
- **Variant A:** Range-focused (280km, sạc tại nhà)
- **Variant B:** Price-focused (240 triệu, rẻ nhất)
- **Variant C:** Warranty-focused (10 năm bảo hành)
- **Variant D:** Technology-focused (pin mới, công nghệ quốc tế)

**Metrics:** CTR, CPC, CVR, CPL

**Test 2: CTA Variations (4 Weeks)**
- **Variant A:** "Đặt lái thử miễn phí"
- **Variant B:** "Báo giá trong 24h"
- **Variant C:** "So sánh ngay hôm nay"
- **Variant D:** "Tải brochure miễn phí"

**Metrics:** Conversion rate, cost per conversion

**Test 3: Pinned vs. Unpinned (4 Weeks)**
- **Variant A:** Brand name pinned position 1
- **Variant B:** No pinning, let Google optimize

**Metrics:** CTR, impression share, auction insights
